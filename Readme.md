# 🌐[S.A.Nafis](https://nafissiddiky2.github.io/Portfolio/) - Personal Portfolio

A modern, responsive personal portfolio website showcasing my skills, certifications, projects, and contact information.

## 🚀 Live Demo

Visit the live website: [S.A.Nafis Portfolio](https://nafissiddiky2.github.io/Portfolio/)

## 📋 Features

- **6 Main Sections:**
  - 🏠 **Home** - Profile picture and introduction with social links
  - 📜 **Certifications** - Professional certificates and achievements
  - 🛠️ **Skills** - Technical skills in Cyber Security, Web Development, Problem Solving, and Graphic Design
  - 🖼️ **Photo Gallery** - Project photos and event memories
  - 🎓 **Education** - Academic background and qualifications
  - 📞 **Contact** - Multiple ways to get in touch

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Dynamic Navigation** - Smooth page loading without refresh
- **Mobile-Friendly** - Hamburger menu for smaller screens
- **Fast Loading** - Static HTML with minimal JavaScript

## 🛠️ Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling, animations, and responsive design
- **JavaScript** - Dynamic page loading and navigation
- **Font Awesome** - Icons throughout the website
- **GitHub Pages** - Hosting and deployment
